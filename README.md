# Wallpaper App

This is a full-stack application with a Node.js backend and React + Tailwind frontend.

## Structure
- `backend/`: Node.js, Express, MongoDB
- `frontend/`: React, Vite, Tailwind CSS

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
